> **Developer note:** This file contains stubs (section headings + TODOs) for the four remaining shop category pages. Each stub follows the same section structure as `docs/15-shop-category-page-template.md`. Fully written copy for each category should be added before build begins. ---

## How to Complete a Stub

1. Copy the full template from `docs/15-shop-category-page-template.md`.
2. Replace all `[CONDITION]`, `[CONDITION_TAG]`, and `[PLACEHOLDER]` tokens.
3. Write Section 1 (Hero) and Section 2 ("This is for you if…") in the brand voice — warm, clinical-meets-compassionate, lived-experience framing.
4. Populate Section 3 (Curated Support Options) with condition-relevant products (max 6 cards).
5. Update all GHL order form names and tags using the naming convention in `docs/15-shop-category-page-template.md`.
6. Replace this stub file entry with a dedicated file (e.g. `docs/18-shop-category-endometriosis.md`) when copy is finalised.
